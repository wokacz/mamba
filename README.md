# mambaPy CLI

mambaPy is a CLI written in Go that wraps the mamba package manager. It is intended to be a drop-in replacement for the conda package manager.

## Installation

```bash
go install github.com/wokacz/mambaPy/cmd/mambaPy@latest
```

## Usage

```bash
mamba --help
```
